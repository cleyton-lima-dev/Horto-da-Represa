# 🌿 Horto da Represa – Site Institucional

Projeto desenvolvido com foco em apresentar produtos e serviços de forma profissional, moderna e funcional para uma empresa real do segmento de plantas ornamentais.

🔗 **Acesse o site:**
https://cleyton-lima-dev.github.io/Horto-da-Represa/

---

## 📌 Sobre o Projeto

O **Horto da Represa** é um site catálogo criado para resolver um problema real: a ausência de uma presença digital estruturada para exposição de produtos e serviços.

O objetivo foi construir uma interface clara, moderna e acessível, permitindo que clientes conheçam os produtos e entrem em contato diretamente via WhatsApp.

---

## 🚀 Funcionalidades

* ✅ Exibição de produtos por categorias
* ✅ Integração direta com WhatsApp para vendas
* ✅ Seção institucional (Sobre a empresa)
* ✅ Apresentação de serviços com carrossel de imagens
* ✅ Mapa integrado com localização da empresa
* ✅ Layout responsivo (adaptado para celular)
* ✅ Interface moderna e focada em conversão
